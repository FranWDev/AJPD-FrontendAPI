Museo Virtual

¡Bienvenido al recorrido virtual interactivo! Explora nuestro jardín botánico utilizando el visor 360 de abajo.

Para navegar, busca los puntos de desplazamiento (iconos circulares) e interactúa con ellos para moverte entre las diferentes escenas. Puedes arrastrar la imagen para mirar a tu alrededor y utilizar el zoom para apreciar los detalles.

Se recomienda pulsar el botón de pantalla completa () y rotar el dispositivo a posición horizontal para una mejor experiencia.
